Learning - Programming In C

Introduction

Introduction to C Programming

C is a general-purpose, procedural programming language that was developed by Dennis Ritchie at Bell Labs in the early 1970s. It was designed to be a system programming language and has since become one of the most widely used programming languages of all time.

Why Learn C?

There are several reasons why learning C can be beneficial:

  • Widely Used: C is used in a wide range of applications, including operating systems, embedded systems, and system software.
  • Efficiency: Programs written in C are typically fast and efficient due to its low-level features and direct access to hardware.
  • Portability: C code can be compiled to run on various platforms with minimal modifications, making it highly portable.
  • Foundation for other languages: Many other programming languages, such as C++, Java, and Python, are influenced by or built upon C, so learning C provides a strong foundation for learning other languages.
  • Understanding of computer architecture: C exposes programmers to low-level concepts such as memory management and pointers, which helps in understanding computer architecture and how programs interact with hardware.

first program

Basic Structure of a C Program

Every C program consists of one or more functions. The main function is the entry point of the program, where execution begins. Here's the basic structure of a C program:

            
#include <stdio.h>

int main() {
    // Statements
    return 0;
}

In the above program:

  • #include <stdio.h>: This line includes the standard input/output library, which provides functions like printf and scanf.
  • int main(): This is the main function of the program. It's where the execution of the program begins.
  • // Statements: These are the statements or instructions that perform the desired tasks of the program.
  • return 0;: This statement indicates the successful completion of the program and returns an exit status of 0 to the operating system.

Writing Your First C Program

Let's write a simple "Hello, World!" program in C:

            
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Save the above code in a file with a .c extension, for example, hello.c.

Compiling and Running Your Program

To compile and run your C program:

  1. Open a terminal or command prompt.
  2. Navigate to the directory where your .c file is located.
  3. Run the executable file generated by the compiler.

And then run the executable:

./hello

variable and data-types

Variables and Data Types

In C programming, variables are used to store data that can be manipulated or processed by the program. Each variable has a data type, which determines the type of data it can store and the operations that can be performed on it.

Data Types in C

C provides several basic data types, including:

  • int: Integer data type (e.g., 42, -10, 0)
  • float: Floating-point data type (e.g., 3.14, -0.5, 2.0)
  • double: Double-precision floating-point data type (e.g., 3.14159, -123.456)
  • char: Character data type (e.g., 'A', 'b', '$')
  • void: Represents the absence of data type
  • bool: Boolean data type (not available in standard C, but commonly used in C99 and later versions)

C also supports modifiers, which can be used with data types to modify their behavior or storage size. Some common modifiers include:

  • short: Shortens the storage size of an integer (e.g., short int)
  • long: Lengthens the storage size of an integer or floating-point number (e.g., long int, long double)
  • unsigned: Specifies that a variable can only hold non-negative values (e.g., unsigned int)

Variable Naming Conventions

When naming variables in C, it's important to follow certain conventions:

  • Variable names can consist of letters, digits, and underscores.
  • Variable names must begin with a letter or underscore.
  • Variable names are case-sensitive (e.g., count and Count are different variables).
  • Choose meaningful and descriptive names for variables to make your code more readable.
  • Follow a consistent naming convention, such as camelCase or snake_case.

Example

Here's an example program that demonstrates the use of additional data types and modifiers:

                        
            #include <stdio.h>
            
            int main() {
                short int distance = 100;
                long double pi = 3.141592653589793238;
                unsigned int score = 95;
            
                printf("Distance: %hd\n", distance);
                printf("Pi: %.15Lf\n", pi);
                printf("Score: %u\n", score);
            
                return 0;
            }

This program declares variables to store distance (short integer), the value of pi (long double), and score (unsigned integer), initializes them with values, and then prints their values to the console.

operator

Arithmetic Operators

Arithmetic operators perform mathematical operations on operands. Examples include:

  • Addition (+): Adds two operands.
  • Subtraction (-): Subtracts the second operand from the first.
  • Multiplication (*): Multiplies two operands.
  • Division (/): Divides the first operand by the second.
  • Modulus (%): Returns the remainder of the division.

Example:

                int a = 10, b = 3;
                int sum = a + b; // sum will be 13
                int difference = a - b; // difference will be 7
                int product = a * b; // product will be 30
                int quotient = a / b; // quotient will be 3
                int remainder = a % b; // remainder will be 1

Relational Operators

Relational operators compare the values of two operands. Examples include:

  • Equal to (==): Returns true if the operands are equal.
  • Not equal to (!=): Returns true if the operands are not equal.
  • Greater than (>): Returns true if the first operand is greater than the second.
  • Less than (<): Returns true if the first operand is less than the second.
  • Greater than or equal to (>=): Returns true if the first operand is greater than or equal to the second.
  • Less than or equal to (<=): Returns true if the first operand is less than or equal to the second.

Example:

                int x = 5, y = 10;
                if (x == y) {
                    // This condition won't be met
                }
                if (x != y) {
                    // This condition will be met
                }
                if (x > y) {
                    // This condition won't be met
                }
                if (x < y) {
                    // This condition will be met
                }

Logical Operators

Logical operators are used to perform logical operations. Examples include:

  • Logical AND (&&): Returns true if both operands are true.
  • Logical OR (||): Returns true if at least one of the operands is true.
  • Logical NOT (!): Returns true if the operand is false, and false if the operand is true.

Example:

                int p = 1, q = 0;
                if (p && q) {
                    // This condition won't be met
                }
                if (p || q) {
                    // This condition will be met
                }
                if (!q) {
                    // This condition will be met
                }

Bitwise Operators

Bitwise operators perform operations on individual bits of operands. Examples include:

  • Bitwise AND (&): Performs a bitwise AND operation.
  • Bitwise OR (|): Performs a bitwise OR operation.
  • Bitwise XOR (^): Performs a bitwise exclusive OR operation.
  • Bitwise NOT (~): Performs a bitwise NOT operation.
  • Left shift (<<): Shifts the bits of the first operand to the left by the number of positions specified by the second operand.
  • Right shift (>>): Shifts the bits of the first operand to the right by the number of positions specified by the second operand.

Example:

                unsigned int a = 5, b = 3;
                unsigned int result_and = a & b; // result will be 1
                unsigned int result_or = a | b; // result will be 7
                unsigned int result_xor = a ^ b; // result will be 6
                unsigned int result_not = ~a; // result will be 4294967290 (assuming 32-bit integers)
                unsigned int result_left_shift = a << 1; // result will be 10
                unsigned int result_right_shift = a >> 1; // result will be 2

Assignment Operators

Assignment operators are used to assign values to variables. Examples include:

  • Assignment (=): Assigns the value of the right operand to the left operand.
  • Compound assignment (+=, -=, *=, /=, %=, &=, |=, ^=, <<=,>>=): Combines an arithmetic or bitwise operation with assignment.

Example:

                int x = 5;
                x += 3; // Equivalent to x = x + 3, x will be 8

Ternary Operator

The ternary operator (? :) is a conditional operator that evaluates an expression and returns one of two values depending on whether the expression is true or false.

Syntax: condition ? value_if_true : value_if_false

Example:

                int x = 5, y = 10;
                int max = (x > y) ? x : y; // max will be 10

control flow

IF-ELSE

The `if-else` statement is a fundamental control flow structure in C programming. It allows your code to make decisions based on conditions, executing different blocks of code depending on whether the condition is true or false.

Syntax

                if (condition) {
                    // code to execute if condition is true
                } else {
                    // code to execute if condition is false (optional)
                }

Here, `condition` is any expression that evaluates to true (non-zero) or false (zero). The code blocks within the `if` and `else` (if present) are executed based on the condition's evaluation.

Examples

1. Checking Even/Odd Numbers

#include <stdio.h>

int main() {
    int number;

    printf("Enter a number: ");
    scanf("%d", &number);

    if (number % 2 == 0) {
        printf("%d is even.\n", number);
    } else {
        printf("%d is odd.\n", number);
    }

    return 0;
}

This example checks if a user-entered number is even or odd. If the remainder of dividing by 2 is 0 (even), the `if` block executes, otherwise the `else` block prints that the number is odd.

2. Finding the Largest of Three Numbers

#include <stdio.h>

int main() {
    int num1, num2, num3;

    printf("Enter three numbers: ");
    scanf("%d %d %d", &num1, &num2, &num3);

    int largest = num1;  // Initialize largest with first number

    if (num2 > largest) {
        largest = num2;
    }
    if (num3 > largest) {
        largest = num3;
    }

    printf("The largest number is: %d\n", largest);

    return 0;
}

This example finds the largest of three user-entered numbers. It initializes `largest` with the first number, then uses nested `if` statements to check if the second and third numbers are greater. If so, `largest` is updated.

Switch-Case Statement

The switch-case statement in C provides a way to execute different blocks of code based on the value of an expression. It is often used as an alternative to long if-else chains when dealing with multiple possible conditions.

Definition

The switch-case statement evaluates the value of an expression and jumps to the corresponding case label. If a case label matches the value of the expression, the code block associated with that case is executed. The break statement is used to exit the switch statement after a case is matched, preventing fall-through to subsequent cases. Additionally, the default case provides a fallback option if none of the case labels match the expression, ensuring that your program behaves predictably even in unexpected scenarios.

Syntax

            
switch (expression) {
    case value1:
        // Code to execute if expression equals value1
        break;
    case value2:
        // Code to execute if expression equals value2
        break;
    // Additional case labels...
    default:
        // Code to execute if expression doesn't match any case
}

Example

Here's an example demonstrating the usage of the switch-case statement:

            
#include <stdio.h>

int main() {
    int day = 4;

    switch (day) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        case 4:
            printf("Thursday\n");
            break;
        case 5:
            printf("Friday\n");
            break;
        default:
            printf("Weekend\n");
    }

    return 0;
}

In this example, the value of the variable day is compared to different cases inside the switch statement. Depending on the value of day, the corresponding day of the week is printed. Since day is 4, it matches the case labeled 4, and "Thursday" is printed.



Loops in C

Loops are used in C programming to execute a block of code repeatedly. They allow you to automate repetitive tasks and iterate over data structures such as arrays. C provides three main types of loops: the while loop, the for loop, and the do-while loop.

1. While Loop

The while loop executes a block of code repeatedly as long as a specified condition is true. 

                
    while (condition) {
        // Code to execute
    }

The condition is evaluated before each iteration. If it evaluates to true, the code block is executed. If it evaluates to false, the loop terminates.

Example:

                
    #include <stdio.h>
    
    int main() {
        int i = 0;
    
        while (i < 5) {
            printf("%d ", i);
            i++;
        }
    
        return 0;
    }

This program prints numbers from 0 to 4.

2. For Loop

The for loop executes a block of code for a specified number of times.

                
    for (initialization; condition; increment/decrement) {
        // Code to execute
    }

The initialization is executed once before the loop starts. The condition is evaluated before each iteration. If it evaluates to true, the code block is executed. After each iteration, the increment/decrement statement is executed. If the condition evaluates to false, the loop terminates.

Example:

                
    #include <stdio.h>
    
    int main() {
        for (int i = 0; i < 5; i++) {
            printf("%d ", i);
        }
    
        return 0;
    }

This program also prints numbers from 0 to 4.

3. Do-While Loop

The do-while loop is similar to the while loop, but it always executes the code block at least once, even if the condition is false.

                
    do {
        // Code to execute
    } while (condition);

The condition is evaluated after each iteration. If it evaluates to true, the loop continues. If it evaluates to false, the loop terminates.

Example:

                
    #include <stdio.h>
    
    int main() {
        int i = 0;
    
        do {
            printf("%d ", i);
            i++;
        } while (i < 5);
    
        return 0;
    }

This program also prints numbers from 0 to 4.

Conclusion

Loops are powerful constructs in C programming that allow you to automate repetitive tasks and iterate over data structures. Understanding how to use while, for, and do-while loops effectively is essential for writing efficient and concise code.

Array

Arrays

An array in C is a collection of elements of the same data type, stored in contiguous memory locations. Arrays provide a convenient way to store and manipulate groups of related data items under a single name.

Declaration and Initialization

To declare an array, you specify the type of elements it will hold, followed by its name and the number of elements enclosed in square brackets ([]).

            type arrayName[arraySize];

For example:

            int numbers[5]; // Declares an array of 5 integers
            float temperatures[10]; // Declares an array of 10 floating-point numbers
            char vowels[] = {'a', 'e', 'i', 'o', 'u'}; // Declares and initializes an array of characters

Accessing Elements

Array elements are accessed using their index, which starts from 0 and goes up to size - 1. You can access individual elements using square brackets notation ([]).

            numbers[0] = 10; // Assigns 10 to the first element of the numbers array
            int x = numbers[2]; // Retrieves the value of the third element of the numbers array

Initialization

Arrays can be initialized at the time of declaration or later using a loop or individual assignment. When initializing at the declaration, you enclose the initial values in curly braces ({}).

Example:

            int squares[5] = {1, 4, 9, 16, 25}; // Initializes an array of squares
            for (int i = 0; i < 10; i++) {
                temperatures[i] = 0.0; // Initializes all elements of the temperatures array to 0.0
            }

Multi-dimensional Arrays

C supports multi-dimensional arrays, allowing you to create arrays of arrays. They are declared using multiple sets of square brackets.

            int matrix[3][3]; // Declares a 3x3 matrix

Array Size and Bounds Checking

C does not perform bounds checking on array accesses, meaning you can access or modify elements outside the bounds of the array, leading to undefined behavior. It's essential to ensure that array indices stay within the bounds of the array to avoid unexpected results or crashes.
